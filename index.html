Kubernetes the hard way

Linux Academy - Diciembre 2020

Kubernetes es una herramienta poderosa para la orquestación de contenedores. Hay varias formas diferentes de iniciar un clúster de Kubernetes, muchas de las cuales facilitan el proceso al abstraer algunos de los detalles. La guía de código abierto de Kelsey Hightower, Kubernetes the Hard Way, explica cómo arrancar un clúster de Kubernetes sin el uso de instaladores o scripts. Si bien no hay nada de malo en usar instaladores o scripts, este enfoque le brinda una comprensión más profunda de los detalles de Kubernetes. Este curso es una adaptación de la guía Kubernetes the Hard Way. Tiene la intención de guiarlo a través de él y brindarle una explicación adicional de cada paso en el camino. NOTA: Se recomiendan microinstancias para su uso en actividades prácticas.
